  • There has an friendly interface to guide you creat you special needs, all you needed were type few word and choice the machine sctructs and then process it.

机床可选类型

  • 机床类型: 外螺纹 内螺纹 丝锥 蜗杆
  • 修整器类型: vw(外) v(外) xz x_后(内) 液压修整(外)
  • 修整轮类型: 单滚轮/方滚轮 双滚轮
  • 选择齿形: 三角/梯形/双圆弧 双圆弧
  • 是否有二次对刀(外)
  • 是否有自动对刀
    • 自动对刀机构类型: 接近开关 测量头 开关和测头
    • 测头测量时测量孔位置：工件正面 工件背面
  • 是否是磨削中心
    • 是否有U轴
    • 无U轴时外磨砂轮结构：后置 前置
  • 是否有A轴
  • 砂轮驱动器类型: 西门子伺服 变频器/普通电机
  • 修整轮驱动器类型: 西门子伺服 变频器/普通电机
  • 磨削中心外磨砂轮驱动器类型: 西门子伺服 变频器/普通电机

程序生成

  1. 运行 HorizonJuicerCreator
  2. 打开 - 设置 配置源仓库路径及生成路径，源仓库目录层级到 Horizon-Juicer 文件夹即可
  3. 选择 配置机床
  4. 输入 机床信息 及 其他配置
  5. 定义 机床配置
  6. 点击 程序生成
  7. 生成成功后会自动打开所在目录

机床导入

  • 配置好的机床目录内有两个文件夹：NC-Data、System-CF-Card
    • NC-Data 存放 NC 加工程序相关内容：MPF、CMA、DEF，分别拷入系统 NC-Data 对应目录内
    • System-CF-Card 存放 HMI 及系统配置文件，有 user、oem、siemens 三个文件夹，分别拷入系统 System-CF-Card 对应目录内
  • DEF 文件导入机床后激活需要激活才能生效
  • 导入 NC 程序后需要重启 NC，导入界面后需重启 HMI
  • 载入变量表： 诊断 -> NC/PLC变量 -> 载入变量表
  • 进入 custom 主界面点击 VS9 机床配置，点击重置配置初始化机床配置，然后根据需要设置需要启用的功能
  • user 目录内配置文件的功能：
    • NC-PLC 变量表：user/sinumerik/hmi/template/variables/NC_PLC-variates.xml
    • VNC 远程连接： user/system/etc/sinumerikvnc.ini
    • 前置 USB 备份数据： user/sinumerik/nck/extdev.ini
  • HMI 在线帮助：若在线帮助hlp进行了修改，需删除siemens/sinumerik/sys_cache/hmi/目录下hlp文件夹并重启HMI

机床可选配置参数：(需在机床配置界面修改，部分参数根据机床配置自动设置)

磨削相关

  • 螺旋升角使能(0没有/1有) INI[16] ---根据配置自动预设
  • 砂轮类型选择(0模拟量/1伺服) INI[36] ---根据配置自动预设
  • 程序结束砂轮是否停止(0停/1不停转速保持/2不停且低速旋转) INI[79] ---预设为 0
  • 砂轮不停低速旋转转速 TECHNOLOGY[9] --- 外磨预设 1000，内磨预设 5000
  • 界面头数修整参数显示：变量表修改WORK[4] ---预设为 0
  • 恒转速模式：变量表修改INI[144] ---预设为 0
  • 扩展工艺：变量表修改PROCESS[15] ---预设为 0
  • 环形槽激活：变量表修改RING[49] ---预设为 0
  • 三线测量参数显示：变量表修改INI[148] ---预设为 0 外磨不显示
  • 转完A轴不关闭液压：变量表修改INI[158]---预设为 0 没有A轴不显示
  • 工艺界面粗磨量自动计算：变量表修改TECHNOLOGY[233] ---预设为 0
  • 内螺纹螺旋退出方式(0退到中心/1后退固定距离)：变量表修改TECHNOLOGY[234] (需先切换到螺旋退出) ---预设为 0 外磨不显示
  • 内螺纹螺旋退出方式为后退固定距离时的距离 TECHNOLOGY[235] ---预设为 0.2 外磨不显示
  • 磨削中每次修整对磨削位置补偿量界面参数显示：变量表修改DRESSER[58] 补偿量输入正值为坐标轴正向补偿 ---预设为 0
  • 内螺纹头架防护罩自动控制(0无1有)：变量表修改PROCESS[120] ---预设为 0 外磨不显示
  • 程序结束是否关闭修整/对刀键 变量表修改INI[70] (0否/1是) ---预设为 0
  • 是否有自动门 变量表修改PROCESS[52] (0否1是) ---预设为 0
  • 自动门选择(自动门1/自动门2) TECHNOLOGY[59] ---预设为 1
  • 自动门是否检测开关信号(0不检测/1检测) TECHNOLOGY[64] ---预设为 1

修整相关

  • 修整轮类型选择(0模拟量/1伺服) INI[37] ---根据配置自动预设
  • 新砂轮插补修整模式(0抬齿高/1偏半个砂轮宽度) INI[88] ---预设为 0
  • 通用齿形模块：变量表修改DRESSER[114] ---预设为 1
  • 修整时A轴角度调整参数显示：变量表修改INI[154] ---预设为 0 没有A轴不显示
  • 内螺纹新砂轮是否是成型砂轮界面参数显示：变量表修改TECHNOLOGY[58] ---预设为 0 外螺纹不显示
  • 修整轮反向旋转：变量表修改INI[155] ---预设为 0
  • 使用外部齿形程序：变量表修改DRESSER[139]=1 程序名写入变量表：SHAPE_EXTERNAL ---预设为 0
  • 执行修整中每20次重新运行砂轮转速子程序(默认为跳过)：变量表修改INI[111] ---预设为 0
  • 通用齿型程序-插补修整平面选择(0:G17-12几何轴/1:G18-13几何轴/2:G19-23几何轴)：变量表修改DRESSER[56] ---预设为 0
  • 磨削中修整是否提前一刀启动滚轮(0否1是)：变量表修改DRESSER[57] ---预设为 0

对刀相关

  • 是否有自动对刀(0没有/1有) INI[69] ---根据配置自动预设 没有自动对刀则隐藏
  • 有无二次对刀(0没有/1有) INI[78] ---根据配置自动预设 内螺纹设置为0
  • 自动对刀后是否直接磨削(0否1是) INI[90] ---预设为 0 磨削中心设置为0隐藏此行
  • 对刀模式(zc插补0/静止对刀1) INI[109] ---预设为 0
  • 内螺纹自动对刀测头测量位置(1工件后/-1工件前) TOOL_SET[43] ---根据配置自动预设 除内螺纹测头对刀外其他配置隐藏此参数
  • 自动对刀测头伸出缩回没有检测开关：变量表修改INI[157] ---预设为 0 没有自动对刀则隐藏
  • 自动对刀测量模式检测(0两次/1一次) 变量表修改INI[159] ---预设为 0没有自动对刀则隐藏
  • 首件工件对刀c起始角度(0:当前c起始角/1:0度对刀) 变量表修改TOOL_SET[56] ---预设为 0 没有自动对刀则隐藏

磨削中心相关

  • 外圆砂轮类型选择(0模拟量/1伺服/2异步电机) INI[91] ---根据配置自动预设
  • 磨削中心只磨内磨时结束不返回外磨Z安全位置：变量表修改PROCESS[118] ---预设为 0
  • 磨削中心外圆/端面加工时头架旋转方向 变量表修改TECHNOLOGY[230] (0正转/1反转) ---根据配置自动预设
  • 磨削中心外磨修整器是否液压控制 变量表修改PROCESS[50] (0否1是) ---根据配置自动预设
  • 磨削中心外磨砂轮修整时头架停止转动并移动到固定角度 变量表修改DRESSER[54] (0否1是) ---预设为 0
  • 磨削中心外磨砂轮修整时头架停止转动移动固定角度 变量表修改DRESSER[55] 预设为0
  • 程序结束外圆砂轮是否停止(0立刻停/1低速旋转直到程序结束/2低速旋转程序结束也不停) INI[92] ---预设为 0
  • 磨削中心外磨加工完后砂轮低速运行转速 TOOL_SET[6] 预设为 800
  • 外圆在线测量参数显示：变量表修改INI[149] ---预设为 0
  • 在线测量FA速度 外圆在线测量独有 INI[116] ---预设为 0.6
  • 在线测量F3速度 外圆在线测量独有 INI[117] ---预设为 0.4
  • 在线测量F2速度 外圆在线测量独有 INI[118] ---预设为 0.2
  • 在线测量延时 外圆在线测量独有 INI[119] ---预设为 8

隐藏界面激活

  • 参数备份恢复界面开启：变量表修改TECHNOLOGY[56]=1 (custom主界面 VS2)---预设为 0
  • 加工统计界面开启：变量表修改TECHNOLOGY[57]=1 (custom主界面 VS5) ---预设为 0
  • 机械手按钮显示开启：变量表修改TOOL_SET[16]=1 磨削界面 VS1/VS2 按钮 ---预设为 0
  • 螺距变换界面开启：变量表修改TOOL_SET[61]=1 (磨削界面显示) ---预设为 0 外螺纹不显示
  • 自磨夹具界面开启：变量表修改TECHNOLOGY[232]=1 (磨削界面显示) ---预设为 0 外螺纹不显示
  • 内螺纹反向孔倒角界面开启：变量表修改INI[21]=1 (磨削界面显示) ---预设为 0 外螺纹不显示
  • 空运行热机界面开启：变量表修改TECHNOLOGY[55]=1 (磨削界面显示) ---预设为 0 外螺纹不显示

界面部分参数设置限制调整

  • 界面中径调整限制值(默认0表示限制为正负1) 变量表修改TOOL_SET[11] 例如设置 2 表示极限值为：-2 ~ +2 ---预设为 0
  • 工艺界面除粗磨外进给量设置限制值(默认0表示限制为正负0.1) 变量表修改TOOL_SET[12] 例如设置 0.2 表示极限值为：-0.2 ~ +0.2 ---预设为 0
  • 修整界面进给量设置限制值(默认0表示限制为 0-0.1) 变量表修改TOOL_SET[13] 例如设置 0.3 表示极限值为：0 ~ +0.3 ---预设为 0
  • 磨削中心-外圆端面修整进给量设置限制值(默认0表示限制为 0-0.1) 变量表修改TOOL_SET[14] 例如设置 0.3 表示极限值为：0 ~ +0.3 ---预设为 0
