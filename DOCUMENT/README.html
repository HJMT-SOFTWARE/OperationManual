Intro

  • The project design to quick build the machine program which may has different spicificaton.
  • There has an friendly interface to guide you creat you special needs, all you needed were type few word and choice the machine sctructs and then process it.

机床可选类型

  • 机床类型: 外螺纹 内螺纹 丝锥 蜗杆
  • 修整器类型: vw(外) v(外) xz x_后(内) 液压修整(外)
  • 修整轮类型: 单滚轮/方滚轮 双滚轮
  • 选择齿形: 三角/梯形/双圆弧 双圆弧
  • 是否有二次对刀(外)
  • 是否有自动对刀
    • 自动对刀机构类型: 接近开关 测量头 开关和测头
  • 是否是磨削中心
    • 是否有U轴
  • 是否有A轴
  • 砂轮驱动器类型: 西门子伺服 变频器/普通电机
  • 修整轮驱动器类型: 西门子伺服 变频器/普通电机
  • 螺纹磨削工艺界面中,粗磨进给量是否自动计算

程序生成

  1. 运行 HorizonJuicerCreator
  2. 打开 - 设置 配置源仓库路径及生成路径，源仓库目录层级到 Horizon-Juicer 文件夹即可
  3. 选择 配置机床
  4. 输入 机床信息 及 其他配置
  5. 定义 机床配置
  6. 点击 程序生成
  7. 生成成功后会自动打开所在目录

机床导入

  • 配置好的文件夹内有四个文件夹(MPF CMA DEF HMI)
  • 使用AMM连接机床后将对应文件夹内文件导入机床
  • HMI内文件夹proj导入Siemens目录，其他导入oem目录即可
  • DEF文件导入机床后激活
  • 导入程序后重启NC，导入界面后重启HMI
  • NC_PLC-variates.xml文件
    • 导入位置: HMI数据 -> 模板 -> 用户 -> 变量表 或者 user\sinumerik\hmi\template\variables
    • 使用: 诊断 -> NC/PLC变量 -> 载入变量表
  • 若在线帮助hlp进行了修改，需删除siemens\sinumerik\sys_cache\hmi\目录下hlp文件夹，重启HMI
  • 新版本系统默认 vnc 功能没有初始化密码无法访问，需要将 sinumerikvnc.ini 文件放入 user/system/etc 文件夹内，重启系统即可

隐藏功能激活：(需在变量表手动修改)

磨削相关

  • 界面头数修整参数显示：变量表修改WORK[4]=1
  • 恒转速模式：变量表修改INI[144]=1
  • 扩展工艺：变量表修改PROCESS[15]=1
  • 环形槽激活：变量表修改RING[49]=1
  • 三线测量参数显示：变量表修改INI[148]=1
  • 转完A轴不关闭液压：变量表修改INI[158]=1
  • C轴设置为主轴模式：变量表修改INI[130]=1 影响环形槽加工指令: DO MOV -> M3
  • 工艺界面粗磨量自动计算：变量表修改TECHNOLOGY[233]=1
  • 内螺纹螺旋退出方式(0退到中心/1后退固定距离)：变量表修改TECHNOLOGY[234] (需先切换到螺旋退出)
  • 界面中径调整限制值(默认0表示限制为正负1) 变量表修改TOOL_SET[11] 例如设置 2 表示极限值为：-2 ~ +2
  • 磨削中每次修整对磨削位置补偿量：变量表修改DRESSER[70]--正值为坐标轴正向补偿 累计补偿量为DRESSER[71]会在修新砂轮或手动对刀后自动清零

修整相关

  • 通用齿形模块：变量表修改DRESSER[114]=1
  • 修整时A轴角度调整参数显示：变量表修改INI[154]=1
  • 修整轮反向旋转：变量表修改INI[155]=1
  • 使用外部齿形程序：变量表修改DRESSER[139]=1 程序名写入变量表：SHAPE_EXTERNAL
  • 跳过修整中每20次重新运行砂轮转速子程序：变量表修改INI[111]=1

对刀相关

  • 自动对刀测头伸出缩回没有检测开关：变量表修改INI[157]=1
  • 自动对刀测量是否二次慢速检测(0否1是) 变量表修改INI[159]=1
  • 首件工件对刀c起始角度(0:当前c起始角/1:0度对刀) 变量表修改TOOL_SET[56]=1

磨削中心相关

  • 外圆在线测量参数显示：变量表修改INI[149]=1
  • 磨削中心只磨内磨时结束不返回外磨Z安全位置：变量表修改PROCESS[118]=1
  • 磨削中心外圆/端面加工时头架旋转方向 变量表修改TECHNOLOGY[230] (0正转/1反转)
  • 磨削中心外磨修整器是否液压控制 变量表修改PROCESS[50] (0否1是)
  • 磨削中心是否有自动门 变量表修改PROCESS[52] (0否1是)
  • 磨削中心外磨砂轮修整时头架停止转动并移动到固定角度 变量表修改DRESSER[54] (0否1是)
  • 磨削中心外磨砂轮修整时头架停止转动移动固定角度 变量表修改DRESSER[55]

隐藏界面激活

  • 螺距变换界面开启：变量表修改TOOL_SET[61]=1 (磨削界面显示)
  • 自磨夹具界面开启：变量表修改TECHNOLOGY[232]=1 (磨削界面显示)
  • 内螺纹反向孔倒角界面开启：变量表修改INI[21]=1 (磨削界面显示)

B_MACHINE_INI 定义隐藏功能

  • INI[16]--螺旋升角使能(0没有/1有)
  • INI[36]--砂轮类型选择(0模拟量/1伺服/2异步电机)
  • INI[37]--修整轮类型选择(0模拟量/1伺服/2异步电机)
  • INI[69]--是否有自动对刀(0没有/1有)
  • INI[70]--是否有自动门(0没有/1有) 除磨削中心外可设置
  • INI[78]--有无二次对刀(0没有/1有) 外螺纹独有
  • INI[79]--程序结束砂轮是否停止(0停/1不停)
  • INI[90]--自动对刀后是否直接磨削(0否1是) 除磨削中心外可设置
  • INI[88]--新砂轮插补修整模式(0抬齿高/1偏半个砂轮宽度)
  • INI[109]--对刀模式(zc插补0/静止对刀1)
  • TOOL_SET[43]--内螺纹自动对刀测头测量位置(1工件后/-1工件前) 内螺纹独有

H_MACHINE_INI 定义隐藏功能(磨削中心独有)

  • INI[91]--外圆砂轮类型选择(0模拟量/1伺服/2异步电机)
  • INI[92]--程序结束外圆砂轮是否停止(0立刻停/1低速旋转直到程序结束/2低速旋转程序结束也不停)
  • INI[116]--在线测量FA速度 外圆在线测量独有
  • INI[117]--在线测量F3速度 外圆在线测量独有
  • INI[118]--在线测量F2速度 外圆在线测量独有
  • INI[119]--在线测量延时 外圆在线测量独有
